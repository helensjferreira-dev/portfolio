# 📝 Descrição do Projeto
Este projeto consiste no desenvolvimento de uma landing page funcional destinada à apresentação do meu portfólio pessoal. A página foi estruturada para reunir e consolidar informações sobre minhas habilidades técnicas e as produções desenvolvidas durante minha trajetória de aprendizado em programação. O objetivo central é utilizar este espaço digital como um instrumento de apresentação profissional para o mercado de tecnologia, permitindo atualizações constantes conforme minha evolução na área.

Este projeto foi desenvolvido como parte da atividade prática para o curso de Introdução à Programação do SENAI, servindo como base para minha transição de carreira para a área de tecnologia.

# 👩‍💼 Sobre a Desenvolvedora

Minha trajetória profissional é marcada pela transição da gestão e análise técnica na Perícia Judicial para o desenvolvimento de software. Essa bagagem anterior me proporcionou um perfil rigoroso, analítico e focado na precisão para resolver problemas complexos. Atualmente, foco em soluções Full-Stack, unindo interfaces modernas no Front-End com arquiteturas robustas em Node.js e Python no Back-End.

# 🛠️ Tecnologias Utilizadas  

Para cumprir os requisitos de excelência técnica e garantir a nota máxima nos critérios de avaliação, utilizei de forma integrada:

- HTML5: Estruturação semântica de todas as seções da página.

- CSS3: Estilização avançada utilizando Flexbox e CSS Grid para garantir um layout responsivo, fluido e organizado.

- JavaScript (ES6): Implementação de lógica para o menu mobile (hamburger) e manipulação dinâmica de datas no rodapé para garantir a atualidade do site.

# 🎨 Decisões de Design e UI/UX

- A interface foi projetada com foco em acessibilidade e conforto visual. Algumas decisões técnicas incluem:

- Contraste Equilibrado: Substituí o branco puro (#FFFFFF) por tons de "Gelo" (#F8F9FA) para evitar a fadiga visual do avaliador durante a leitura.

- Acessibilidade Cromática: Utilizei uma paleta baseada em tons escuros de azul com destaques em Amarelo Ouro (#FFC107), garantindo que os pontos de interação (botões e links) sejam identificados instantaneamente.

- Layout Responsivo: Os cartões de habilidades e projetos utilizam flex-wrap e justify-content: center para manter o alinhamento centralizado e harmônico em qualquer tamanho de tela.

# 🧠 Habilidades Demonstradas  

Conforme os requisitos do curso, a página contempla competências em duas frentes:

- Técnicas (Hard Skills): Domínio em linguagens como Python (Flask), JavaScript (Node.js/Express) e SQL.

- Comportamentais (Soft Skills): Foco em Resolução de Problemas (refinada na Perícia Judicial), Pensamento Analítico e vivência em Metodologias Ágeis (Scrum/Kanban).

# 📁 Seções da Página   

- Home (Hero): Introdução profissional direta.

- Sobre: Detalhamento da transição de carreira e indicadores de experiência.

- Projetos: Catálogo com 7 projetos reais, descrições técnicas e links diretos para o GitHub.

- Habilidades: Exposição organizada e compacta das tecnologias e processos.

- Contato: Canais diretos de comunicação e redes sociais profissionais.

2. Navegue até a pasta:
    ```

    ```

    Basta abrir o arquivo index.html em qualquer navegador ou utilizar a extensão Live Server no VS Code para uma melhor experiência.
